Quick API Reference

# Get all assets

GET https://codefuturist.github.io/asset-library/catalog/assets.json

# Get images only

GET https://codefuturist.github.io/asset-library/catalog/images.json

# Get asset metadata

GET https://codefuturist.github.io/asset-library/assets/[type]/[category]/[id]/metadata.json
